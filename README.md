# Astranex Defence

A minimalist, high-performance website for Astranex Defence systems.

## Overview

This project is built with pure HTML, CSS, and Vanilla JavaScript to ensure maximum performance and a "bare metal" feel suitable for a defence technology company.

## How to Run

Simply open `index.html` in your web browser. No installation or server is required.

## Features

- **Theme**: Pure black & white, sovereign defence aesthetic.
- **Animations**: Smooth scroll, fade-in elements, and dynamic system status.
- **Responsive**: Fully responsive grid layout for all devices.
- **Typography**: Inter font for clean, sharp executve tone.
